# CLAW Mission MVP - TechResume AI

AI-powered resume optimization service for tech professionals.

## Features
- AI analysis of resumes for ATS compatibility
- Tech-specific keyword optimization
- 24-hour turnaround for beta testers
- Free for first 10 signups

## Deployment
This is a static site deployed via GitHub Pages.

### Pages URL
`https://[username].github.io/claw-mvp/`

### Local Development
1. Clone the repository
2. Open `index.html` in a browser
3. Make changes to HTML/CSS/JS files

## Google Form Integration
The landing page integrates with Google Form for beta signups.

## Email Automation
- Auto-response to beta signups
- 24-hour turnaround notifications
- Feedback collection
