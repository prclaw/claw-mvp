# ResumeCraft AI - Brand Guidelines

## Brand Identity
- **Primary Name:** ResumeCraft AI
- **Backup Name:** CareerCanvas AI
- **Tagline:** "Get 3x More Interviews with AI-Optimized Resumes"
- **Mission:** Transform tech professionals' resumes to pass ATS systems and land more interviews

## Color Palette

### Primary Colors
- **Primary Blue:** `#2563eb` (Main brand color)
- **Dark Blue:** `#1e40af` (Hover states, accents)
- **Light Blue:** `#60a5fa` (Backgrounds, highlights)

### Accent Colors
- **Accent Orange:** `#f59e0b` (CTAs, important elements)
- **Success Green:** `#10b981` (Confirmation, positive)
- **Warning Red:** `#ef4444` (Errors, warnings)

### Neutral Colors
- **Dark Gray:** `#1f2937` (Headings)
- **Medium Gray:** `#4b5563` (Body text)
- **Light Gray:** `#9ca3af` (Secondary text)
- **Background Gray:** `#f8f9fa` (Page background)
- **White:** `#ffffff` (Cards, sections)

## Typography

### Primary Font: Inter
- **Weights:** 400 (Regular), 500 (Medium), 600 (Semi-bold), 700 (Bold)
- **Usage:** Body text, paragraphs, buttons

### Heading Font: Poppins
- **Weights:** 600 (Semi-bold), 700 (Bold), 800 (Extra-bold)
- **Usage:** All headings (h1-h6), hero text, section titles

### Font Sizes
- **h1:** 48px (Desktop) / 36px (Mobile)
- **h2:** 36px (Desktop) / 28px (Mobile)
- **h3:** 24px (Desktop) / 20px (Mobile)
- **Body:** 16px (Desktop) / 14px (Mobile)
- **Small:** 14px (Desktop) / 12px (Mobile)

## Logo Concept
- **Style:** Modern, clean, tech-forward
- **Elements:** AI brain icon + document/paper icon combination
- **Colors:** Primary blue (#2563eb) with accent orange (#f59e0b)
- **Typography:** Poppins Bold for "ResumeCraft", Inter Medium for "AI"

## Visual Style
- **Clean & Professional:** Minimalist design with ample white space
- **Tech-Forward:** Subtle gradients, modern shadows, rounded corners
- **Consistent:** Uniform spacing, consistent button styles
- **Accessible:** High contrast ratios, clear hierarchy

## UI Components

### Buttons
- **Primary CTA:** `#2563eb` background, white text, 8px border-radius
- **Secondary CTA:** White background, `#2563eb` border, `#2563eb` text
- **Hover States:** Darken background by 10%, add subtle shadow
- **Active States:** Scale down to 95%

### Cards
- **Background:** White
- **Border Radius:** 12px
- **Shadow:** 0 10px 25px rgba(0, 0, 0, 0.1)
- **Padding:** 32px

### Forms
- **Input Border:** 1px solid `#d1d5db`
- **Focus Border:** 2px solid `#2563eb`
- **Border Radius:** 6px
- **Padding:** 12px 16px

## Spacing System
- **Base Unit:** 4px
- **Small:** 8px (2 × base)
- **Medium:** 16px (4 × base)
- **Large:** 24px (6 × base)
- **X-Large:** 32px (8 × base)
- **XX-Large:** 48px (12 × base)

## Iconography
- **Style:** Line icons with rounded corners
- **Size:** 24px for standard, 40px for feature icons
- **Color:** Primary blue or accent orange

## Responsive Breakpoints
- **Mobile:** < 768px
- **Tablet:** 768px - 1024px
- **Desktop:** > 1024px

## Voice & Tone
- **Professional:** Clear, concise, expert
- **Confident:** Results-focused, data-driven
- **Helpful:** Supportive, educational
- **Urgent:** Time-sensitive, action-oriented

## Key Messages
1. "Get 3x More Interviews with AI-Optimized Resumes"
2. "Tech-specific resume optimization that passes ATS systems"
3. "24-hour delivery guaranteed"
4. "Money-back guarantee if you don't get more interviews"
5. "Trusted by X tech professionals"

## Implementation Notes
- Use CSS variables for easy theming
- Implement mobile-first responsive design
- Ensure WCAG 2.1 AA accessibility compliance
- Optimize for fast loading (<2 seconds)
- Test across Chrome, Firefox, Safari, Edge
